项目文件：

	simplex.py	实现代码文件

	input.txt	输入数据文件
	
	testdata.txt	多组待测试数据文件

	output.txt	多组测试结果文件

输入文件格式说明：

	1			# 1表示目标函数是max；0表示目标函数是min
	4 5 1   		# 目标函数系数向量
	3   3   		# 约束条件矩阵维数
	3 2 1 18 2         	# 最后一个数字表示约束条件符号，2表示大于等于
	2 1 0 4 1          	# 最后一个数字表示约束条件符号，1表示小于等于
	1 2 0 5 0        	# 最后一个数字表示约束条件符号，0表示等于


注意事项：

	一次只支持输入一组数据

	需要用户手动修改 input.txt

	约束矩阵统一保留两位小数输出

	所有变量x_1, x_2, …, x_n默认非负
	
	建议使用VScode打开本项目的任何文档

存在的一些问题：

	忽略了对问题“是否需要用到两阶段法”的判断
	对所有问题统一使用两阶段法

	表格缩进采用简化方式，不完全对齐
